``object.__dict__``
===================
.. code-block:: python

    >>> class Astronaut():
    ...    def __init__(self):
    ...        self.first_name = 'José'
    ...        self.last_name = 'Jiménez'

    >>> jose = Astronaut()

    >>> jose.__dict__
    {'first_name': 'José', 'last_name': 'Jiménez'}
